# CLIcalc

CLIcalc é um projeto pessoal, desenvolvido como **template base** de várias linguagens de programação, com o objetivo de facilitar o aprendizado de novas linguagens de programação.

A ideia principal é usar este projeto como um **desafio prático**, onde você recria sua estrutura em outra linguagem, como JavaScript, Java, TypeScript, Shell etc, aprendendo conceitos fundamentais como:
- Variáveis
- Tipos de dados
- Operadores 
- Condicionais
- Loops/Iteração
- Funções
- Modularização

Para usar, basta entrar na pasta de alguma linguagem e rodar, verificando suas dependencias.

---
## 📁 Estrutura
Ao recriar o projeto em outra linguagem, siga a seguinte estrutura, como no código acima:

### 1. Main
Deve existir um arquivo principal responsável por:
- Controlar o fluxo do programa  
- Interagir com o usuário  
- Chamar as funções auxiliares  

### 2. Pasta Modules
Deve haver uma pasta chamada modules, com os seguintes arquivos abaixo:

### 2.1. Auxiliary
Responsável por funções auxiliares. Nele, deve haver funções como:
- `validarNumero` - valida entradas numéricas  
- `validarOperacao` - valida a operação escolhida  
- `calcularResultado` - gerencia o cálculo com base na operação  

### 2.2. Operações
Contém as funções matemáticas, como:
- `somar`  
- `diminuir`  
- `multiplicar`  
- `dividir`  
- `expoenciar`  
- `radiciar`

## ⚙️ Regras
Para garantir o máximo aprendizado, algumas regras devem ser seguidas.

### 📌 Pouco uso de bibliotecas externas
- Priorize bibliotecas nativas da linguagem  
- Garanta portabilidade
- Foque na lógica do programa

### 📌 Uso mínimo de IA
- Evite depender de IA, como ChatGPT, Gemini, Claude
- Tente resolver os problemas por conta própria  
- o principal método de aprendizado deve ser forums ou documentação da linguagem em questão

### 📌 Portabilidade essencial
- Compatível com qualquer versão (ou as mais recentes)
- Deve ser compatível em Linux
- Não deve haver diferenças exorbitantes entre sistemas operacionais
